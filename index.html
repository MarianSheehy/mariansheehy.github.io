Bank Broker

A Hub Where Users Can Connect to Various Banks and Generate Reports Using Transaction Data

Marian Sheehy

About the Project

Bank Broker is a proof-of-concept web application for bank reporting capabilities. Current banking applications are dated in their user friendliness. They are limited in their reporting selection and tend to have very restricted access to historic data.

This application will be a user hub with links to the main Irish banks. Users can access all the data in one place, perform data analysis and create more meaningful financial reporting.
